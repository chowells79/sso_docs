Event

Create

Send a POST request to https://api.moxiworks.com/api/event

When creating an event using the Moxi Works platform API, format your data using the following parameters.

Event Creation Examplemoxi_works_agent_idabc123partner_event_idabc982cdf345partner_agent_idzxy987event_subjectLet’s meet for lunchevent_location1234 56th Avesend_remindertrueremind_minutes_before15event_start1456955102event_end1456955602attendeesabc123, xcr898recurringfalsenote<p>Hi, this is an HTML formatted note to be displayed for this meeting</p>

Attribute Type Remark Required
moxi_works_agent_id String The agent’s Moxi Works Platform ID required
partner_event_id String The unique ID of the event in your system required
partner_agent_id String The unique ID of the agent in your system
event_subject String A short description of the event
event_location String A description of the location of the event
send_reminder Boolean Whether a notification reminder should be sent before the meeting
remind_minutes_before Integer How many minutes before the event a reminder should be sent (if it is to be sent)
event_start Integer The POSIX timestamp representation of the start time – local time.
event_end Integer The POSIX timestamp representation of the end time – local time.
attendees String A comma separated list of contacts that have already been added through the Moxi Works Platform API. (use IDs from your system) who are to be included as attendees.
recurring Boolean Whether the event is recurring
note String a note to be included with additional details. This can be HTML formatted.

Update

Send a PUT request to https://api.moxiworks.com/api/event
Moxi Works API will remove data for any attribute not defined within the paramters you send. For instance, if the home_street_address attribute is not passed, the current home_street_address for the contact will be removed from the contact stored in Moxi Works.

Show

Send a GET request to https://api.moxiworks.com/api/event/[partner_event_id]
Attribute Type Remark Required
moxi_works_agent_id String The agent’s Moxi Works Platform ID required
partner_event_id String The unique ID of the contact in your system partner_event_id or moxi_works_contact_id is required
moxi_works_event_id String The unique ID of the event in the Moxi Works Platform partner_event_id or moxi_works_event_id is required

Delete

Send a DELETE request to https://api.moxiworks.com/api/event/[partner_event_id]
Attribute Type Remark Required
moxi_works_agent_id String The agent’s Moxi Works Platform ID required
partner_event_id String The unique ID of the event in your system partner_event_id or moxi_works_event_id is required
moxi_works_event_id String The unique ID of the event in the Moxi Works Platform partner_event_id or moxi_works_event_id is required
